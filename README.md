# BOOK STAGE

### Folder Structure

1. controllers : 사용자의 요청을 받아 필요한 데이터를 레포지토리로 전달
2. repositories : DB에 접근
3. loaders : 모듈 기본 설정
4. routers : 라우팅
5. middlewares : 컨트롤러에 접근하기 전 인증 로직 등을 구현
6. utils : 공통 함수
7. errors : 공통 에러
